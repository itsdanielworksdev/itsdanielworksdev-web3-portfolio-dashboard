# assets/

Place static assets here:

- `avatar.png` — Your profile photo (recommended: 400×400px)
- `resume.pdf` — Your resume for direct download
- `og-image.png` — Open Graph social preview image (1200×630px)
- `logo.svg` — SVG logo variant

Reference them in components via:
```js
import avatar from '../assets/avatar.png'
```
