# itsdanielworksdev Web3 Dashboard

A modern Web3 dashboard built with React and Node.js, designed to display blockchain data, user portfolio information, and interactive Web3 features with a clean and responsive interface.

## Features

- Modern responsive dashboard UI
- Web3 wallet connection
- Blockchain data integration
- User portfolio display
- Clean developer-friendly architecture

## Tech Stack

Frontend:
- React
- Next.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Tools:
- Git & GitHub
- APIs
